Цей Telegram-бот допомагає користувачам вивчати іноземні слова, зберігати їх у власному словнику, тренуватися та отримувати пояснення від штучного інтелекту (Google Gemini).

Також у проєкті є **Адмін-панель** для перегляду статистики та бази даних користувачів.


Функціонал

 Telegram Бот (`bot.py`)
* Додавання слів: Зберігає слово, його переклад та мову. Автоматичний переклад через Google Translate.
* Тренування: Режим перевірки знань (картки).
* Статистика: Відображає кількість вивчених слів та рівень користувача.
* Слово дня: Випадкове слово зі словника для повторення.
* AI Помічник: Інтеграція з Google Gemini для пояснення слів та контексту їх використання.
* Захист від спаму: Обмеження частоти запитів.

Адмін-панель (`admin.py`)
* Зручний графічний інтерфейс (Tkinter).
* Перегляд списку користувачів та їхньої активності.
* Перегляд словника кожного користувача.
* Сортування даних та підсвічування активних користувачів.

Встановлення та запуск

1. Клонування репозиторію
Завантажте файли проєкту в одну папку.

2. Встановлення бібліотек
Встановіть необхідні залежності за допомогою команди:
```bash
pip install -r requirements.txt
````

3. Налаштування API ключів

Відкрийте файл `bot.py` і вставте свої ключі:

```python
TELEGRAM_BOT_TOKEN = "ВАШ_ТОКЕН_ТЕЛЕГРАМ"
GEMINI_API_KEY = "ВАШ_КЛЮЧ_GEMINI"
```





Як користуватися

Запуск бота

```bash
python bot.py
```

Знайдіть бота в Telegram і натисніть `/start`.

Запуск адмін-панелі

```bash
python admin.py
```

Відкриється вікно з таблицями користувачів та слів. Дані оновлюються автоматично кожні 5 секунд.




Технології

  * **Python 3.11**
  * **Aiogram 3.x** (Telegram API)
  * **SQLite** (База даних)
  * **Google Gemini API** (Штучний інтелект)
  * **Tkinter** (GUI для адмін-панелі)
